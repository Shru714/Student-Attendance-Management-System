================================================================================
FIELD NAMES IN DEMO DATA
================================================================================

CLASSES:
- id: Unique identifier
- className: Class name (PHP, Java, etc.)
- class_section: Section (A, B, etc.)
- year: Year (1, 2, 3)
- academic_year: Academic year (2025-2026)

TEACHERS:
- id: Unique identifier
- name: Teacher's full name
- email: Teacher's email
- teacherId: Unique teacher ID (TCH001, etc.)
- contactNo: Contact number (10 digits)
- phone: Phone number
- years: Array of years taught [1, 2, 3]
- classIds: Array of class IDs [1, 2, 3]

STUDENTS:
- id: Unique identifier
- student_name: Student's full name
- email: Student's email
- rollNumber: Auto-generated roll number
- classId: Class ID
- address: Student's address
- student_contact: Student's contact number
- parent_contact: Parent's contact number
- password: Auto-generated password

ATTENDANCE:
- id: Unique identifier
- classId: Class ID
- studentId: Student ID
- studentName: Student's name
- rollNumber: Student's roll number
- date: Date (YYYY-MM-DD format)
- time: Time (HH:MM format)
- status: 'present' or 'absent'
- markedAt: Timestamp when marked

================================================================================
CLEARING DEMO DATA
================================================================================

If you want to clear demo data and start fresh:

METHOD 1: Browser DevTools
1. Press F12 to open DevTools
2. Go to "Application" or "Storage" tab
3. Click "Local Storage"
4. Find your domain
5. Right-click and select "Clear"
6. Refresh the page

METHOD 2: Using Console
1. Press F12 to open DevTools
2. Go to "Console" tab
3. Type: localStorage.clear()
4. Press Enter
5. Refresh the page

================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: Demo data button not showing
SOLUTION:
1. Hard refresh: Ctrl+Shift+R
2. Login as Admin
3. Look for "ðŸ“Š Load Demo Data" button on Dashboard

ISSUE: Classes not showing in dropdown
SOLUTION:
1. Load demo data first
2. Hard refresh: Ctrl+Shift+R
3. Login as Teacher
4. Go to Mark Attendance
5. Click on class dropdown

ISSUE: Students not loading
SOLUTION:
1. Make sure demo data is loaded
2. Select a class from dropdown
3. Students should auto-load
4. If not, check browser console for errors

ISSUE: Attendance not saving
SOLUTION:
1. Make sure at least one student is marked
2. Check that date and time are selected
3. Try hard refresh: Ctrl+Shift+R
4. Try again
